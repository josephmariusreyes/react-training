# React Essentials - TypeScript Version

This is the TypeScript version of the React Essentials project demonstrating outputting list data.

## Setup

Install dependencies:

```bash
npm install
```

## Development

Run the development server:

```bash
npm run dev
```

## Build

Build for production:

```bash
npm run build
```

## Preview

Preview the production build:

```bash
npm run preview
```

## TypeScript Features

This version includes:
- Full TypeScript type definitions for all components
- Type-safe props interfaces
- Type declarations for image imports
- Strict TypeScript configuration
- Type-safe state management with typed useState hooks
